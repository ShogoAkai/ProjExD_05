# ゲーム のタイトル
よけろ！！
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
線と玉を避ける

## ゲームの実装
###共通基本機能
* 基本行動に関するクラス
* 線とドットの親クラス
* 垂直の線に関するクラス
* 水平の線に関するクラス
* ドットに関するクラス
### 担当追加機能
* 経過時間の表示、10秒後に落ちてくる玉が視認しずらくなり、難易度が上がる。
